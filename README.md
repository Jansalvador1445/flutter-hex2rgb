## Usage
### Mobile Devices
```
git clone https://github.com/Jansalvador1445/flutter-hex2rgb
cd flutter-hex2rgb
flutter pub get
flutter run
```

### Web
```
git clone https://github.com/Jansalvador1445/flutter-hex2rgb
cd flutter-hex2rgb
flutter pub get
flutter run -d chrome
```

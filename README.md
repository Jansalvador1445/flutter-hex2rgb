# hex2rgb

A color conversion from Hex to RGB using Flutter.

## Prerequisites
- [Flutter](https://flutter.dev/docs/get-started/install)

## Usage
```
git clone https://github.com/Jansalvador1445/flutter-hex2rgb
cd flutter-hex2rgb
flutter pub get
flutter run
```
